
# LetMeQuestion

Application to learn and share knowledge with others.

## Tech Stack

**Client:** React

**Server:** Firebase

  
## Run Locally

Clone the project

```bash
  git clone https://github.com/bllackdev/letmequestion.git
```

Go to the project directory

```bash
  cd /letmequestion/
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```
